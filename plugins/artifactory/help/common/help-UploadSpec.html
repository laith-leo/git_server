To upload files, you need to create a File Spec.
The File Spec schema for uploading files is as follows:
{
    "files": [
        {
            "pattern": "[Mandatory]",
            "target": "[Mandatory]",
            "props": "[Optional]",
            "recursive": "[Optional, Default: 'true']",
            "flat" : "[Optional, Default: 'true']",
            "regexp": "[Optional, Default: 'false']"
        }
    ]
}
