To download files, you need to create a File Spec.
The File Spec schema for downloading files is as follows:
{
    "files": [
        {
            "pattern" or "aql": "[Mandatory]",
            "target": "[Optional]",
            "props": "[Optional]",
            "recursive": "[Optional, Default: true]",
            "flat" : "[Optional, Default: false]"
        }
    ]
}
